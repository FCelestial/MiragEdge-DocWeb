# 锐界幻境文档站 ✰ MiragEdge Docs

## 📖 项目介绍

### 介绍
锐界幻境（MiragEdge）是基于高版本Minecraft（Java 1.21.11）的跨平台互通服务器，支持Java版与基岩版玩家同服游玩。本仓库是服务器官方文档库，涵盖**玩家指南**、**开发协作**、**版本日志**等核心内容，旨在帮助玩家快速上手、开发者高效协作，同时记录服务器的成长历程。

锐界幻境致力于打造一个开放创新的服务器平台，为玩家提供纯净的游戏体验。我们注重技术创新、社区建设和内容创造，努力在数字世界中为每位玩家打造属于自己的幻境。

## 🌟 核心特性

### 🔧 技术架构
- **跨平台互通**：基于Geyser实现Java版与基岩版无缝连接
- **高性能支持**：采用Purpur服务端核心 + Waterfall负载均衡架构
- **硬件保障**：i5-14600KF超频物理机 + 64GB高频内存 + 企业级SSD存储
- **数据安全**：每日4次本地增量备份 + 每周完整克隆备份 + 每月云端备份

### 🎮 游戏特色
- **创新玩法**：独家轻RPG系统、星露谷、附魔等多维度玩法
- **真实季节**：自定义季节系统，无需安装资源包即可体验四季变化
- **丰富经济**：完善的经济系统与玩家工坊机制
- **多样化食物**：丰富的自定义食物系统，带有多样化效果

### 🌐 社区理念
- **纯净体验**：拒绝强制付费，坚持公益运营
- **7×24小时管理**：几乎全天候管理团队值守
- **跨次元社交**：游戏内互动与QQ等玩家社区深度联动
- **共创文化**：鼓励玩家参与内容创作与服务器建设

### 🛡️ 安全保障
- **反作弊体系**：混合型强力反作弊（客户端验证+服务端行为分析）
- **网络优化**：多线路负载均衡，电信/移动/联通BGP优化
- **内容保护**：文档站对接长亭雷池防火墙，核心内容加密存储
- **自动白名单**：QQ绑定自动白名单系统，保障账号安全

## 🏗️ 技术栈
本项目采用现代前端技术栈构建：
```bash
Framework: VitePress 1.6.4
Language: TypeScript 5.9.3
Build: Node.js with optimized dependencies
Visualization: Mermaid, Canvas-confetti
```

### 主要依赖
- **VitePress**: 现代静态站点生成器
- **Mermaid**: 图表和流程图渲染
- **Canvas-confetti**: 粒子动画效果
- **Axios**: HTTP客户端
- **@octokit/rest**: GitHub API客户端
